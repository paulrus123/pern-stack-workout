## To Run the Backend Locally

Navigate to the project directory:

```bash
cd ./my-node-project/
```
Start the server:

```bash
node ./server.js
```

## To Run the Frontend Locally

Navigate to the project directory:

```bash
cd ./my-react-app/
```
Start the react app:

```bash
npm run start
```

Note: this assumes postgres db is available for backend to communicate to.
